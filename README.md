# Aplicación interactiva para la resolución numérica de EDP de segundo orden

Aplicación desarrollada en **MATLAB App Designer** para la resolución numérica de ecuaciones diferenciales parciales (EDP) de segundo orden mediante métodos de diferencias finitas.

## Módulos implementados

- EDP Elípticas
- EDP Parabólicas
- EDP Hiperbólicas

La aplicación permite configurar el dominio, las condiciones iniciales y de contorno, los parámetros numéricos y la solución exacta (cuando está disponible). Además, incorpora herramientas para la visualización de resultados mediante gráficas, tablas y métricas de desempeño, incluyendo errores numéricos y tiempos de ejecución.

## Archivos del proyecto

**Archivo principal**
- `SolverEDP2_101.mlapp`

**Archivos complementarios**
- Figuras identificadoras utilizadas por la interfaz de la aplicación.

## Instrucciones de uso

1. Descargue el archivo principal y todos los archivos complementarios del repositorio.
2. Guarde todos los archivos en una misma carpeta.
3. Abra `SolverEDP2_101.mlapp` en **MATLAB**.
4. Ejecute la aplicación y configure el problema de interés mediante la interfaz gráfica.
